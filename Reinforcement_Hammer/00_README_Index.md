# GA_Design_Docs (Index & 운영 가이드)

이 폴더는 ‘강화 컬렉션 자동전투(+999)’ 기획 문서를 **일관되게 관리**하기 위한 패키지입니다.

## 문서 역할(소스 오브 트루스)
- **GDD(02)**: 규칙/UX/운영의 “결정 문장” = 제품 기준
- **Ledger(03)**: 경제·밸런스의 “숫자 기준” = 튜닝 기준
- **Working(01)**: 아이디어/논의/메모 = 작업 공간(결정 전)

## 운영 원칙(추천)
1) 논의는 Working(01)에서 시작
2) 결정되면 GDD(02)로 올리고, 숫자는 Ledger(03)로 올린다
3) 매 작업 단위마다 CHANGELOG와 DECISIONS를 갱신한다(짧게)
